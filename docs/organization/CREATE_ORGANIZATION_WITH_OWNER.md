## â“ Common Questions

### Q: Can I create without assigning an owner?
**A**: Yes, but it's recommended to assign one. You can add the owner later using "Manage Roles".

### Q: Can I change the owner later?
**A**: Yes! Go to "Manage Roles" â†’ Add new admin â†’ Change old admin to staff.

### Q: What if I can't find the user?
**A**: Make sure they've registered in the system first. Check their email is correct.

### Q: Can one person be owner of multiple organizations?
**A**: Yes! A user can be admin of multiple organizations.

### Q: What permissions does the owner get?
**A**: All 19 permissions - full control over the organization.

### Q: Can I have multiple owners?
**A**: No, only one Administrator per organization. But you can have one Vice Administrator.

## ğŸ†˜ Troubleshooting

### Problem: Button says "Manage Communities" is disabled
**Solution**: Only Provincial Minister and Vice Provincial can access this. Contact your administrator.

### Problem: Can't find user in the list
**Solution**: 
1. Check if they've registered
2. Try searching by email
3. Refresh the page
4. Contact system administrator

### Problem: Error when saving
**Solution**:
1. Check all required fields are filled
2. Verify user is selected
3. Try again
4. If persists, contact support

### Problem: Owner not showing as admin
**Solution**:
1. Go to "Manage Roles"
2. Check if they're listed as Administrator
3. If not, click "Add Member" and assign them
4. Select "Administrator" role
