# Super Admin File Access - Quick Start Guide

## üöÄ Quick Start (3 Steps)

### Step 1: Navigate to Organization
Click **Organization** in the sidebar (Building icon)

### Step 2: Select a Community
Click **View Details** on any friary, school, or parish card

### Step 3: View Files
Click the **Files & Documents** tab

That's it! You can now view, upload, and manage files for that organization.

## üéØ Common Tasks

### View Files for a Friary
1. Go to **Organization** in sidebar
2. Find the friary (e.g., "St. Francis Friary")
3. Click **View Details**
4. Click **Files & Documents** tab
5. Browse, search, or filter files

### Upload a File to an Organization
1. Go to **Organization** ‚Üí Select community ‚Üí **Files** tab
2. Click **Upload File** button
3. Select file from your computer
4. Enter file name and category
5. Click **Upload File**

### Find All Files from a School
1. Go to **Files** in sidebar
2. Click **Organization** dropdown
3. Select the school (e.g., "Sacred Heart School")
4. View filtered files

### Download a Document
1. Find the file (any method above)
2. Click the **eye icon** to view
3. Or click **download icon** to download directly

### Delete a File
1. Find the file
2. Click the **trash icon**
3. Confirm deletion

## üí° Pro Tips

### Tip 1: Use Organization Dashboard for Focused Work
When working with a specific community, use the Organization Dashboard ‚Üí Files tab. It's cleaner and more focused.

### Tip 2: Use File Manager for Cross-Organization Work
When you need to see files from multiple organizations, use the File Manager with filters.

### Tip 3: Use AI Assistant for Quick Searches
Ask AI: "Show me all financial reports from last month" - it searches across all files you have access to.

### Tip 4: Add Organization When Uploading
Always select an organization when uploading files. This makes them easier to find and manage later.

### Tip 5: Use Categories Consistently
Use the same categories across all organizations:
- **Reports** - Financial and activity reports
- **Documents** - General documents
- **Forms** - Application and request forms
- **Policies** - Rules and guidelines
- **Minutes** - Meeting minutes
- **Correspondence** - Letters and communications

## üÜò Quick Troubleshooting

### Can't see files?
- Check if you're logged in as super admin
- Verify the organization has files uploaded
- Try refreshing the page

### Can't upload files?
- Check your role permissions
- Verify file size is under 50MB
- Make sure you selected an organization

### Files not showing in search?
- Check your filters (Category, Organization)
- Try clearing search and filters
- Refresh the page
