### Add More Members
```
1. Find organization in list
2. Click "Manage Roles"
3. Click "Add Member"
4. Select role (Vice Admin or Staff)
```

### Role Options
- **Administrator** (1 person) - Full control
- **Vice Administrator** (1 person) - Almost full control
- **Staff** (unlimited) - Operational access
- **Viewer** (unlimited) - Read-only
